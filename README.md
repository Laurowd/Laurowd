### 💣 Projeto em Destaque
**[Bomb Defuser](https://github.com/Laurowd/Bomb-Defuser)** Um minigame interativo estilo arcade construído em C puro (bare-metal) para a placa de desenvolvimento BitDogLab (RP2040).
* **Destaques Técnicos:** Controle assíncrono de uma matriz WS2812B via PIO e DMA, renderização de display OLED customizada via I2C, leitura analógica por ADC, e controle de estados (FSM) por interrupções de hardware (IRQ).
